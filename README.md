# Project for my gas (CO, NH3 and NO2) and temperature data logger

This project uses a Raspberry Pico W to read sensors MICS-6814 to get values of CO, NH3 and NO2 concentration in air(not calibrated yet) and a
DHT11 to get temperature and humidity values and print them in to an OLED SSD1306. Also send those values through WiFi to a server as plaintext
